[[[ 리눅스

# 개념
UBUNTU : 리눅스 운영체제 
VMWARE : 리눅스 운영체제를 설치할 가성머신, 가짜로 서버를 만듬
PUTTY : 운영체제 접속프로그램

# 입력
cd : 이동 (. 현재위치, .. 상위폴더로 이동)
pwd : 현재 위치
ctrl + c : 강제종료 ( * 복사가 아님 ) 
clear : 지우기
su - : root로 계정변경
ls : 해당 폴더 내의 모든 파일들
ls -a : 숨겨진 파일까지 전부 확인하기
ls - l : 파일의 개수
ls - al : 파일의 개수 + 숨긴 파일목록 확인
* 리눅스의 옵션은 섞어치기가 가능
* 리눅스의 #는 주석
cat 파일명 : 들어가지 않고 파일의 내용 확인가능
rm 파일명 : 파일 삭제
rm -i a : 삭제시 물어봄 (y/n으로 대답)
touch : 파일 생성

# alias
alias : 별명 등록된 목록 확인
alias rm='rm -i' : 별명등록, rm을 요청하면 rm -i처럼 기능하라(직접등록)
alias ll='ls -alF' : 상세히 보기 별명등록(기본옵션)
vi .bashrc : alias 삭제되지 않게 직접 파일에 등록하기


# 리눅스 명령모드, 입력모드, 콜론모드(마지막줄 모드)
vi 파일명  : 명령모드로 파일 조회
* vi는 사용시 없는 파일도 만들어준다
a, i, o 	: 명령모드에서 입력모드로

esc 	: 입력모드에서 명령모드로
:    	: 콜론모드로
* 모드에서 완전히 나오는 법 : 명령모드에서 콜론모드에서 저장명령 내리기
- wq : 저장하고 닫기(콜론모드) / q : 그냥 나가기 / w 파일명 : 다른이름으로 저장
/ q! : 저장안하고 나오기
아무런 키 : 콜론모드에서 명령모드로

# 모드별 키
- 명령모드
HLJK : 방향키 이동
$ : 맨 뒤
^ : 맨 앞
G : 마지막
u(소문자) : 입력모드에서 수정한 내용 전부 취소시키기


# 경로
절대경로 : /로 시작
상대경로 : /로 시작안함

# 폴더구조
/(루트) - home - guru(일반유저)

# 리눅스 관리자 계정 : root (super user 줄여서 su)
최초 root 계정의 비밀번호는 세팅해야한다
관리자계정 비밀번호 설정 : sudo passwd root

# 접속
* putty 접속시 root 접속은 막혀있다
* guru로 접속한 다음 root 계정으로 변경
방법 : su - : root로 계정변경
