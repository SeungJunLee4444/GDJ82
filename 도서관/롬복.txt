# 롬복 에너테이션 종류
@Setter : setter 메서드
@Getter : getter 메서드
@NoArgsConstructor : 디폴트 생성자
@AllArgsConstructor : 매개변수 생성자
@ToString : 문자열 변환
@Data : getter, setter, tostring
@Builder : 빌더타입
