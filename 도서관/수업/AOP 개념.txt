# AOP 
- 개념
관점지향 프로그래밍
흩어진 ASPECT(관심사)를 모듈화하는 프로그래밍 기법
특정 로직의 공통, 핵심적인 관점을 모듈화한다
- 수업 : 컨트롤러에 요청된 이벤트를 실행하기 전후에 공통된 작업을 모듈화해서 처리
트랜잭션 처리가 가능하다

- 조인포인트 : 컨트롤러의 모든 메서드를 말한다
- 포인트컷 : 조인포인트에서 모듈화를 위해 선정한 메서드들을 말한다

- 장점 : 기존코드에 비침투적, 로직의 특정구역에 일일히 들어가야할 코드를 전부 aop에서 처리
가능하기 때문

- 기능
로그인전 체크
트랜잭션
