# undefined  : 변수가 선언되었으나 값이 저장된 적이 없는 상태
ex) 존재하지 않는 배열의 요소를 읽으려할때

# null 	     : null, 의도적으로 값을 비운 상태

# ' ' 	     : 공백, 빈 문자열, 값이 존재함, 메모리 차지

* var 키워드로 선언한 변수는 무조건 undefined로 초기화된다
* null은 이전에 참조하던 값을 더이상 참조하지 않는 다는 의미이며, 
자바스크립트는 빈 메모리공간을 가비지 컬렉션으로 처리해 메모리를 제거한다

# 둘의 차이
undefined는 자바스크립트가 변수를 초기화할때, null은 개발자가 의도적으로 변수를 초기화할때
사용한다


# java
null : 어떠한 값으로도 초기화되지 않은 것, 초기화된 데이터, 메모리상에 존재x
""    : 하나의 스트링, 빈값을 메모리에 할당한 행동
