# 편집서비스 요청
1) edit 서비스 : 
(1) 해당 번호를 파라미터로 전달하여 db에서 해당 번호에 관한 정보를 가져온다
(2) 편집창에 얻어온 정보를 뿌린다

(2) modify 서비스 : 
가져온 한개의 정보(ex 게시판)에서 전달할 파라미터 정보
+ 해당 인원을 의미하는 번호(ex boardNo)를 같이 전달해줘야한다
(쿼리문에서 번호가 동일한 경우를 조건으로 수정하기위해)

* 이를 위한 조건 : 파라미터로 식별번호(boardNo)를 전달해줘야한다
방법 (1) 요청 경로에 직접 파라미터를 추가한다 : location.href를 사용하는 경우
방법 (2) input type hidden 속성을 이용한다 : form 태그를 사용하는 경우
