[2] html 개념

1. HTML
=> HYPER TEXT MARKUP LANGUAGE
=> 하이퍼 텍스트(링크를 만드는 텍스트)를 만드는 언어


2. 태그의 특징
=> 정해진 태그를 사용
=> <> 안에 작성
=> 소문자로 작성
=> 계층구조로 작성(부모 - 자식)


3. 태그의 종류
1) 자식태그	: <여는태그></닫는태그>
2) 나홀로 태그	: <태그>, <태그/>


4. 태그의 기본 구성
1) html:5 선언 : <!DOCTYPE html>, 문서 타입선언			& html:5
2) <html> 태그 : 전체내용 포함
3) <head> 태그 : 헤더
4) <body> 태그 : 본문

5. EMMIT
1) 홈페이지 기본태그 자동완성 
- 형태 : html:5 또는 ! + 엔터

2) 부모+자식태그 자동완성
- ex) ul>li*5
=> ul을 부모로 li을 자식으로 5개 태그 만들기


6. 레벨
- 태그의 적용범위

1) block
- 적용범위를 한 줄만큼 차지
- 주로 구조와 관련된 태그					& 블록 : 구조
- 종류 : 
<h>
<p>
<pre>
<div>
<ul> <ol> <li> 
<dl> <dt> <dd>
<table> 


2) inline		
- 적용범위를 텍스트만큼 차지
- 주로 텍스트와 관련된 태그					& 인라인 : 텍스트
- 줄바꿈 x : <br>로 해결					& inline은 줄바꿈x
- 종류 :
<br>
<img> <a> 
<span>
<strong> <em> <ins> <sup> <sub>
