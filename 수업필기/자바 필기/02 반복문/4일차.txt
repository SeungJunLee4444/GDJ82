[1] 반복문
1. for
=> 반복횟수가 파악 가능할 때
1) 형태 
(1) for( 초기문, 조건문, 증감문) {
실행문
}

(2) 중첩 for문
=> 여러 변수를 동시에 출력하고 싶을 떄
ex) ~월 ~일, 구구단, ~일차 ~교시 등등




2) 진행순서
초기문 => 조건문 => 실행문 => 증강문 => 조건문(반복)~

3) 특징
(1) for문의 변수값
=> for문의 로컬변수와 필요에 따라 밖에서 선언된 다른 변수가 있다
=> 이 둘의 혼동을 유의할 것



(2) 반복문에서 실행문은 for문 내부에서 돌리면 최종값이 나오기까지의 모든 값이
for문 밖에 두면 최종값만 산출된다
(로컬변수든 다른 변수든)
 

2. while 
=> 반복횟수를 파악 못할때, 무한 반복
=> 정보를 확인할 때 사용

1) 형태 (형태는 if와 같으나, 반복문이라 조건이 충족되면 다시 처음으로 돌아감)
=> while(조건문) {
실행문
}

2) 특징
1) while(true) 문 : 무한루프문 * 

3. 기타
1) break

=> for 중첩의 경우 label을 사용하면 바깥 for문으로 탈출이 가능

2) continue (안중요함)
=> 반복문에서 도중에 특정조건에 해당하면 해당동작을 중단하고 다시 처음으로 돌아감
=> 특정 조건을 값에서 제외시키기 위한 수단
