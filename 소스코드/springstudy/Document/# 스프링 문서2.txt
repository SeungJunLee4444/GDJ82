# 포워드 이동과 리다이렉트 이동
포워드 이동 : get요청, jsp이동, select문
리다이렉트 이동 : post 요청, insert, update, delete 문

# crud 컨트롤러
전체조회 
- 포워드 이동 / model에 속성 저장 / 별도의 매개변수(파라미터)x

상세조회 : 
- 포워드 이동 / 별도의 매개변수(파라미터) 전달, 주로 번호

추가 : 
(1) 추가창 이동
- 포워드 이동, 매개변수 없음
(2) 추가 요청
- 리다이렉트 이동, 매개변수로 파라미터 전달, 

편집 : 
(1) 편집화면 이동
- 포워드 이동 / 해당 열람 번호를 매개변수로 전달 / 조회한 해당 객체를 model에 속성 저장 (상세문과 같은 형태)

(2) 편집 요청
- 리다이렉트 이동, 매개변수로 객체 전달, 상세화면으로 돌아오는 경우 해당 객체의 번호를 파라미터로 반환경로에 추가

삭제 : 
- 리다이렉트 이동, 매개변수로 객체 번호 전달


* 컨트롤러에서 경로를 반환할경우 각 메서드의 반환타입은 string
반환하지않고, 서비스에서 응답하는 경우는 void
응답한 값을 전달해주는 ajax의 경우 전달해주는 값의 타입이다



# @Component

- 컨트롤러      : @controller 
- 서비스        : @service
- 레파지토리    : @repository / @mapper(mybatis)
